## Prerequisites
Requires ganache. To install it (on Ubuntu 20.04) run:
```
sudo apt install npm
npm install -g ganache-cli
```
Installation is verified with:
```
ganache-cli --version
```
Which returns:
```
Ganache CLI v6.12.2 (ganache-core: 2.13.2)
```
